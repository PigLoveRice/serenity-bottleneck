# Serenity 产业链瓶颈方法论

本方法论用于 A 股产业投研，不构成投资建议。

## 核心框架

1. 先从产业结构而不是股价开始：BOM 清单、供应链地图、客户关系、产能、认证进度、标准演进。
2. 寻找"下一层瓶颈"：当市场只看到表层需求时，向上游材料、设备、封装、光互联、电力、执行器等环节拆解。
3. 形成高质量先验：产业位置、TAM 拐点、客户验证、政策/资本开支、竞争格局、估值错配。
4. 在不完全信息下行动：只在期望值足够且可证伪路径清楚时进入候选，不追求 100% 确定。
5. 持有期间用证据更新：财报、公告、订单、产能、管理层表述、客户映射、价格/量能。
6. 负面证据同等重要：替代路线、客户自研、稀释、管理层失信、竞争扩产、拥挤交易都会降低置信度。
7. 动态轮动到后验最高处：资金和研究精力流向最活跃、最可验证、风险收益更好的瓶颈。

## 瓶颈定位四问

对产业链的每一个环节，依次回答：

1. **最难扩产？** 工艺复杂、设备壁垒高、良率难做 → 扩产周期 > 18 个月
2. **最难替代？** 认证周期长、客户切换成本高、专利护城河 → 新进入者需要 3+ 年
3. **最容易被忽视？** 市值小、业务冷门、无研报覆盖、不在热门概念榜单上
4. **寡头格局？** 全球只有 2-3 家能做，或国内只有 1-2 家有量产能力

满足任意两项即为"瓶颈环节"。

## 趋势优先产业排序

1. 时代主线优先：先判断资本开支、利润、产能和估值会往哪个产业方向迁移，再讨论个股。
2. 拥挤度需要重定义：早期主线的资源集中不等于泡沫；真正危险的是未来增量已被充分定价。
3. 下一层瓶颈优先：从整机/模块继续下钻到芯片、材料、设备、连接器等卡点。
4. 预期差不是冷门：高关注方向仍可能有业绩超预期，冷门方向也可能没有催化。
5. 产业逻辑要量化：需求量 → 供给量 → 缺口 → 价格 → 单位利润 → 公司弹性。

## 证据分级标准

| 等级 | 定义 | 示例 |
|------|------|------|
| P0 | 直接可追溯的一手证据 | 年报、招股书、监管公告、客户/供应商公开材料 |
| P1 | 多源归纳的强信号 | 券商研报（交叉验证）、行业标准、大厂供应商名单 |
| P2 | 框架推断或间接信号 | 新闻、产业链逻辑推理、社交媒体线索（仅作线索） |

## 输出约束

- 每个候选必须给出瓶颈定位、证据等级、风险标记和覆盖缺口。
- 社交媒体只能作为线索，不得单独支持高置信度结论。
- 高置信度必须有 P0 主来源和至少一个独立 P1/P2 交叉验证。
- 不得给出买卖建议或仓位比例，只返回研究和风险框架。
- 输出是研究候选，不是投资建议。
